Label Tool by LWZ

Working Modes

Part I

This application supports two modes: bounding box mode and painting mode.

(1) Bounding box mode records bounding boxes of the objects on the target image together with their classes. This mode is usually for object detection.

(2) Painting mode enables you to draw masks on the image. And it's usally used for segmenation.

You can switch between these two modes by modifying config.cfg under the folder configure

GUI explanation

Part II

Menu buttons

(1) Using file menu, you can select the folder contains the images your want to label by pressing "Select image folder", and choose the folder to save your results by pressing "Saving image to...".

(2) The second button on the menu bar will save current status of labeled image, and display the next image in the folder. And you can also do this by pressing "N" of your keyboard.

(3) The third button enables you to go back to the previous images while keeping the bounding boxes on the previous image. And the shortcut for this function is "P". Note, currently, this button only works under the "bounding box mode."

(4) The fourth button can undo the operations on the current image. Also you can press "Z" to do this.

(5) The fifth button provides more options.

The options (number of classes, bbx_color, Ruler_color, Move mode) are used for bounding box mode. The options (Pen width,Move mode) are used for painting mode

(5.1) Adjust the number of class to display better.

(5.2) Ajust the bounding box color using BBx_color option

(5.3) Ajust the rulder color using ruler_color option

(5.4) Move mode: if you choose to move original images, those labeld original images will be moved from its original folder to the target folder saving the result. The copy mode will keep the image where it exists. The default mode keeps the original mode

(6) The sixth button will show this document.

Saved results

Part III

Two folders will be created under the root directory for saving results

They are arranges as

The folder, with prefix Labeld_image, contains the results of the images which you draw bounding boxes or masks on them. And the suffix is the time print

The folder, with prefix Unlabeld_image, contains the results of the images which you do not draw bounding boxes or masks on them. And the suffix is the time print
